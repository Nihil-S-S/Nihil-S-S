# Hi, I'm Nihil 👋

**Embedded Software Engineer** at Jio Platforms Limited, Bangalore  
NIT Calicut — ECE 2019

I work on Embedded Linux BSP, boot chain firmware, and Yocto-based build systems for NXP i.MX9x SoC platforms. Currently deepening my skills in Linux device drivers, kernel internals, and real-time Linux.

---

## 🔧 What I work on

```
Power-on → ROM code → SPL → TF-A BL31 → U-Boot → Linux Kernel → Userspace
               ↑ I own this entire chain
```

- **BSP Development** — NXP i.MX91 (Cortex-A55 AArch64), full boot chain bring-up
- **Yocto Project** — custom layers, BitBake recipes, machine configs, SDK generation
- **U-Boot** — SPL, boot scripts, env vars, TFTP boot, DTB manipulation
- **Linux Kernel** — kernel 6.6 LTS, device tree authoring, driver integration, pinmux
- **Security** — ARM TF-A BL31, OP-TEE TrustZone, secure boot chain
- **OTA Updates** — RAUC A/B redundant update system, slot management, rollback logic
- **Connectivity** — NXP SDIW610 WiFi/BT (SDIO), Gigabit Ethernet (RGMII, IEEE 1588 PTP)
